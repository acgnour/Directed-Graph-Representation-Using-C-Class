# Directed Graph Representation Using C++ Class

## Overview
This project demonstrates how a directed graph can be represented using C++ classes and an adjacency matrix.

A directed graph is a graph where every edge has a direction from one vertex to another.

The program allows the user to:
- Create a graph
- Add directed edges
- Display the adjacency matrix

---

## Data Structure Used
Directed Graph

The graph is represented using:
- Vertices (nodes)
- Directed edges
- Adjacency matrix

---

## Features
- Create graph using a class
- Store graph connections in an adjacency matrix
- Add directed edges
- Display graph structure
- Demonstrate graph directionality

---

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
